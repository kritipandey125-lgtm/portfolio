# 💼 Personal Portfolio Website

This is my personal developer portfolio website built to showcase my skills, projects, experience, and contact information.

It is designed with a modern UI, smooth animations, and responsive layout for all devices.

---

## 🌐 Live Demo
https://kritipandey125-lgtm.github.io/portfolio/

---

## ✨ Features
- Fully responsive design (mobile, tablet, desktop)
- Dark / Light mode toggle
- Smooth scroll animations
- Interactive project section with "View More"
- Services section with expandable details
- Skills section with organized categories
- Qualification timeline (Education & Experience tabs)
- Contact form UI

---

- HTML5
- CSS3
- JavaScript
